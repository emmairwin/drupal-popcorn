drupal-popcorn
==============

Drupal Popcorn Development 
Currently this is a proof of concept Module , defining popcorn fields as field types for Drupal nodes;
dependant on video.js
ie:

video id = video uploaded w/ video.js

popcornwikipedia
=>videostarttime
=>videoendtime
=>wikipediaurl

popcorngooglemap
=>long
=>lat
=>videostarttime
=>videoendtime
